Usar um <H1> significa que esse é um assunto principal.

Já usar o <H2> é um sub-assunto do <H1>.

O <H3> também é um sub-assunto do <H2> e assim sucessivamente, até o <H6>.

Existem no total 6 tags de Hierarquia, que vão do <H1> até o <H6>.

Formatação de Texto

a tag <strong> serve para deixar um texto em negrito. Não para dar forma, mas sim para dar sentido ao texto. É bom lembrar que, antigamente, algumas tags eram usadas para dar uma forma nos textos escritos, entretanto, hoje em dia isso foi deixado de lado e as tags passaram a serem usadas para dar sentido. A forma dos sites está totalmente ligada ao CSS e o HTML ao sentido.

Outra tag importante é a tag <em> que serve para deixar o texto em itálico. relembrando, esta tag é para dar ênfase ao texto, como quando queremos dizer o nome de uma rua e usamos o itálico para enfatizar essa informação.

Se quisermos marcar um texto, utilizamos a tag <mark> que marca a frase ou palavra selecionada com a cor amarela. Bom se você quiser usar outra cor você pode usar o style do CSS e escrever assim:
<style>
mark
{
background-color
}
</style>

Agora se quiser deixar a letra menor, voce pode usar a tag <small>, que faz que o texto diminua um pouco, por exemplo: "Hello,World!"

Podemos também deletar um texto. Por exemplo, algum desenvolvedor fez um site, só que ao analisar o site percebeu uma frase fora de contexto ou algo desnecessário você pode usar a tag <del>

Outra informação que podemos citar é o texto inserido. Sua tag é <ins>

As tags <sup> e <sub> são tags para dizer que algum texto está subscrito ou sobrescrito

Se por algum acaso você quiser mostrar um código em um site, precisa saber que existem duas tags que se adequam corretamente ao seu texto de código para ocorrer um ajuste nas linhas, para assim ficar de fácil entendimento. Essas tags são o <code> e o <pre>. Podemos observar um texto com e outro sem a tag:

Agora um com as tags <code> <pre>:
